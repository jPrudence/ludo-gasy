# Ludo Gasy

Implementation of the logic of the Ludo game in JavaScript & Vue.js
## Live demo

[https://ludo-gasy.netlify.app](https://ludo-gasy.netlify.app)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
